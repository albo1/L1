# Level 1 Medical Products Website

A modern, responsive website for Level 1 Medical Products - providing portable oxygen systems, stairlifts, and medical equipment rentals.

## 🚀 Project Overview

This website showcases medical equipment and services including:
- Portable Oxygen Concentrators
- Straight Stairlifts
- Medical Equipment Rentals
- Company Information

## 📁 Project Structure

```
level1/
├── index.html              # Homepage
├── about.html              # About Us page
├── portable-oxygen.html    # Portable Oxygen Systems page
├── stairlifts.html         # Stairlifts page
├── rentals.html            # Rentals page
├── css/
│   └── style.css          # Main stylesheet
├── js/
│   └── script.js          # JavaScript functionality
└── images/                 # Image assets (to be added)
```

## 🛠️ Setup & Installation

1. **Clone or download** this project to your local machine
2. **Add images**: Place all product images in the `images/` folder
3. **Open in browser**: Simply open `index.html` in your web browser

No build process or dependencies required - this is a static HTML website!

## 📝 Customization Guide

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #0066cc;      /* Main brand color */
    --secondary-color: #003d7a;    /* Darker accent */
    --accent-color: #00a8e8;       /* Bright accent */
}
```

All content is in the HTML files. Simply edit the text within the HTML tags to update:

- Paragraphs: `<p>` tags
